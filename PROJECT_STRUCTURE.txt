  /api/casas/{id}      - Get house by ID
  /api/profesores      - List all professors

  /api/estudiantes     - List all students

  /api/mascotas        - List all pets
  /api/mascotas/{id}   - Get pet by ID
  /api/asignaturas     - List all subjects
  /api/asignaturas/{id}- Get subject by ID
